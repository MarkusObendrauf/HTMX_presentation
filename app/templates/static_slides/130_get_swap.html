Get, Swap, Target

  • hx-get will issue a GET to the specified URL
    hx-get="/example"
  • hx-target allows you to target an element for swapping
    hx-target="closest div"
  • hx-swap allows you specify how the response will be swapped relative to the target
    hx-swap="afterend"
